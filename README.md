# Optuna MLP

A demo of Optuna using MLP (multilayer perceptron)

# Prereqirement
* Python version >= 3.7.3
* SQLite https://www.sqlite.org/index.html

# Installation
```bash
git clone git@github.com:yellowshippo/optuna_mlp.git
cd optuna_mlp
pip3 install -r requirements.txt
```

# Execution
```bash
python3 main.py
```
